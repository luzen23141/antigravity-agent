## 使用說明

<details>
<summary><b>新增帳戶</b></summary>

<br />

1. 點擊 **登入新帳戶** 按鈕
2. 確認後，當前帳戶資料將被清除
3. 在自動開啟的 Antigravity 中登入新帳戶，帳戶資訊將自動儲存

</details>

## 常見問題

- **macOS 用戶無法安裝？**
  由於應用程式未簽署，macOS 可能會阻止執行。請查看 [macOS 安裝指南](macos-installation.zh-CN.md)。

- **Linux 用戶如何安裝？**
  我們推薦使用 Flatpak 版本以獲得最佳體驗。請查看 [Flatpak 安裝指南](flatpak-install.zh-CN.md)。
