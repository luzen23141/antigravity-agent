## Usage Guide

<details>
<summary><b>Add New Account</b></summary>

<br />

1. Click the **Login New Account** button
2. After confirmation, the current account data will be cleared
3. Log in to the new account in the automatically opened Antigravity, and the account information will be saved automatically

</details>

## FAQ

- **Can't install on macOS?**

- **How to install on Linux?**
  We recommend using the Flatpak version for the best experience. Please see the [Flatpak Installation Guide](flatpak-install.en.md).
