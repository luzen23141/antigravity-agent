## 使用说明

<details>
<summary><b>添加新账户</b></summary>

<br />

1. 点击 **登录新账户** 按钮
2. 确认后，当前账户数据将被清除
3. 在自动打开的 Antigravity 中登录新账户，账户信息将自动保存

</details>
